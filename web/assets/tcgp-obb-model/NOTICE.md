TCGP OBB crop model
===================

Source: https://github.com/1vcian/Pokemon-TCGP-Card-Scanner

The model files in this directory are the TensorFlow.js export from that
project's `pokemon-tcgp-card-scanner/public/model/` directory, commit
`37692ae2f44d2ecf76caf4ee9c1305bbad298f44`.

The upstream model metadata identifies it as an Ultralytics YOLO11n-OBB model
with `license: AGPL-3.0 License (https://ultralytics.com/license)`.
